# 📝 Bloco de Notas Full Stack

Aplicação Full Stack desenvolvida com React e Spring Boot para gerenciamento de notas.

## 🚀 Tecnologias

- React
- Bootstrap
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## ✨ Funcionalidades

- Criar notas
- Listar notas
- Editar notas
- Excluir notas

## ▶️ Como executar

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
