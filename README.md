# 📝 Bloco de Notas Full Stack

Aplicação Full Stack desenvolvida com React e Spring Boot para gerenciamento de notas.

## 🚀 Tecnologias

- React
- Bootstrap
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

## ✨ Funcionalidades

- Criar notas
- Listar notas
- Editar notas
- Excluir notas

## ▶️ Como executar

### Backend

```bash
cd backend
./mvnw spring-boot:run
```

### Configuração do banco

Antes de iniciar o backend, configure a variável de ambiente da senha do PostgreSQL:

```powershell
$env:DB_PASSWORD="sua_senha_do_postgresql"
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
